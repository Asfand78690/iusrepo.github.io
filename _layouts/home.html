IUS is a yum repository that provides newer versions of select software for RHEL and CentOS.

Modern software for Enterprise Linux

RHEL typically prioritizes stability over adding new features. This results in a great operating system, but there are times when users are willing to sacrifice some of that stability in order to gain access to features in newer software.

No automatic replacement of stock packages

Our packages use different names than their stock equivalents, and will never obsolete a stock package name. This ensures that a stock package will never be unexpectedly replaced by an IUS package just because the repository is enabled.

Follows upstream versions

Our packages track upstream branches. If the upstream follows semantic versioning, that means our package will include the major version number in the name and will be updated to the latest of that major version (but never the next major version).
